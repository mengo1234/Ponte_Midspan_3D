# Ponte Midspan 3D

Viewer locale e due STL estratte da `local_midspan_repaired.fea`.

- `local_midspan_repaired_FE_surface_100mm.stl`: mesh FE esatta, non remeshata.
- `local_midspan_repaired_FE_surface_100mm_print_repaired.stl`: versione watertight per slicer/stampa.
- `index.html`: viewer 3D con selezione modello e download di entrambi gli STL.
